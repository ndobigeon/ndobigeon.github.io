Hyperspectral unmixing


Our activities deal with spectral unmixing of hyperspectral data.

For these models...

  • LMM : Linear Mixing Model
       (the most popular model)
  • GBM : Generalized Bilinear Model
       (a nonlinear model to address multiple scattering effects)
  • PPNMM : Polynomial Post-Nonlinear Mixing Model
       (a flexible nonlinear model)
  • RLMM : Robust Linear Mixing Model
       (a group-sparse model generalizing the LMM)
  • NCM : Normal Compositional Model
       (statistical model to address spectral variability)
  • PLMM : Perturbed Linear Mixing Model
       (deterministic model to address spectral variability)

under these scenarios...

  • supervised : inversion step (spectral signatures are known)
  • unsupervised : joint estimation of spectral signatures and abundances

and these contexts

  • online (on-the-fly processing)
  • off-line (batch processing)
  • partially asynchronous distributed processing

using these methods...

  • Bayesian estimation and MCMC sampling
  • (scaled) gradient optimization
  • block coordinate descent algorithm (with majorization-minimization steps)
  • ADMM algorithm
  • PALM algorithm
